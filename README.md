# SOC Detection Lab

#Overview

This repository showcases a hands-on Security Operations Center (SOC) lab built using Splunk Enterprise. It demonstrates attack simulation, log collection, SIEM detection engineering, dashboard creation, and security event investigation through realistic attack scenarios.

The objective is to strengthen practical SOC skills by simulating attacks in a controlled environment and developing detections using Splunk Search Processing Language (SPL).

---

#Lab Environment

SIEM | Splunk Enterprise 10.4.1 |
Attacker | Ubuntu Linux + Hydra + Nmap |
Victim | Ubuntu Linux |
 Operating System | Ubuntu 26.04 LTS |
Log Source | /var/log/auth.log |
